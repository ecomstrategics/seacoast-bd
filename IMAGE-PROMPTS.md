# Seacoast Building & Design — GPT-Image-2.0 Prompt Pack

This pack contains copy-paste prompts for **ChatGPT Images 2.0 / GPT-Image-2** to generate every missing image on the Seacoast Building & Design site.

## How to use

1. Open the ChatGPT app (any tier with GPT-Image-2 / ChatGPT Images 2.0 image generation).
2. For each image below, copy the prompt inside the fenced code block and paste it as a new message.
3. Generate. If you don't love it, ask for a variation ("same composition, different roof color" / "tighter framing" / etc.) rather than rewriting from scratch.
4. Download the result as `.png` or `.webp`. Save it using the **suggested filename** under the heading.
5. Drop saved files into `repos/ecomstrategics/seacoast-bd/public/images/` (subfolders noted per image). Hal will wire them into the components and replace the `ImagePlaceholder` calls.

## Style guardrails (already baked into every prompt below)

- Photorealistic, editorial-quality contractor marketing photography.
- Southwest Florida residential/coastal setting — Fort Myers, Naples, Sarasota, Cape Coral, etc. Lush tropical landscaping, palms, suburban stucco-and-tile architecture.
- Bright Florida natural light unless the scene is explicitly stormy.
- Shot on a full-frame DSLR aesthetic, balanced exposure, sharp focus, natural color grading. No HDR halos, no over-saturation, no plastic skin.
- No visible text, logos, signage with readable letters, watermarks, or AI giveaways. If signage is present in scene, keep it generic and softly blurred.
- People in scenes: real, professional, age 30-55, in neutral work clothing or polo + jeans, no branded uniforms.
- Aspect ratio is stated explicitly in each prompt.
